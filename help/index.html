Introduction

This documents contain documentation for PartSeg program.

This is tool for interactive segmentation data from high resolution microscope images. User can set:

  • Threshold type (Lower threshold | Upper threshold)
  • Threshold value (integer number)
  • Minimum object size (integer number) - every object smaller than this value will be hidden
  • Using of noise reduction by gauss smooth and radius for this gauss (real positive number)
  • Modify final segmentation with drawing and erasing

Result of segmentation can be save as project file. This file contains information about value of each parameters.
